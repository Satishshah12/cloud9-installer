# 🚀 Viper Cloud9 IDE Installer

One-click Cloud9 IDE installer for Ubuntu VPS.

Deploy a complete browser-based development environment using Docker and LinuxServer Cloud9.

---

## ✨ Features

✅ Cloud9 IDE  
✅ Docker Based  
✅ PHP Development Environment  
✅ Node.js 22 LTS  
✅ Python3  
✅ Git  
✅ Composer  
✅ Persistent Workspace  
✅ Auto Restart Container  
✅ Random Password Generator  
✅ Ubuntu VPS Support
